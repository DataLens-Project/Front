# Data Lens 최종 실사용 체크리스트 (2026-04-18)

## 1) 수정/원본 버튼 전환 UI 확인

### 목표
- 원본 행 데이터 버튼 오른쪽에 수정된 행 데이터 버튼이 보이고,
- 버튼 클릭 시 같은 테이블 영역에서 데이터가 전환되어야 한다.

### 절차
1. AnalysisLab에서 CSV 파일 업로드
2. 질문 입력: 나이 없는 행은 무작위로 채워줘 최대60
3. 데이터 테이블 탭으로 이동
4. 상단 버튼 영역 확인
   - 원본 행 데이터
   - 수정된 행 데이터 (원본 버튼 오른쪽)
5. 원본 행 데이터 버튼 클릭 후 내용 확인
6. 수정된 행 데이터 버튼 클릭 후 내용 변경 확인

### 캡처 포인트
- 캡처 A: 두 버튼이 나란히 보이는 상태
- 캡처 B: 원본 행 데이터 버튼 선택 상태
- 캡처 C: 수정된 행 데이터 버튼 선택 상태

### 기대 결과
- 버튼 2개가 모두 노출
- 버튼 전환 시 표시 행 데이터가 실제로 달라짐

## 2) PDF 한글 깨짐 확인

### 목표
- PDF 다운로드 시 한글이 물음표(????)로 깨지지 않고 읽혀야 한다.

### 절차
1. 분석 완료 후 인사이트 리포트 탭 이동
2. PDF 다운로드 클릭
3. 다운로드된 PDF 열기
4. 다음 텍스트 확인
   - 데이터 렌즈 분석 리포트
   - 추천 기법
   - 설명/인사이트 문장(한글 문장)

### 캡처 포인트
- 캡처 D: PDF 뷰어에서 한글 제목과 본문이 정상 렌더링된 화면

### 기대 결과
- 본문 한글이 깨지지 않음

### 서버 운영 메모
- 서버 폰트 고정 권장:
  - 환경변수 DATALENS_PDF_FONT_PATH 설정
  - 예: /app/fonts/NanumGothic.ttf

## 3) 대용량 행 렌더링(가상 스크롤) 확인

### 목표
- 10,000행 이상에서도 스크롤이 즉시 반응하고 브라우저가 멈추지 않아야 한다.

### 절차
1. 10,000행 이상 CSV 업로드
2. 데이터 테이블 탭 진입
3. 표를 빠르게 스크롤 업/다운
4. 컬럼 헤더에서 정렬, 필터, 리사이즈 동작 확인

### 캡처 포인트
- 캡처 E: 스크롤 중 헤더 고정 상태
- 캡처 F: 필터 팝업 적용 화면

### 기대 결과
- 스크롤 버벅임 없이 반응
- 헤더가 상단에 고정
- 정렬/필터/리사이즈 동작 정상
